## ğŸ’» Installing Wazuh Agent on Ubuntu Endpoint


**Step 2: Add Wazuh repository**

```bash

```

**Step 3: Update package list**

```bash
apt-get update
```

**Step 4: Install the agent**

```bash

```
